O que é HTML

HTML significa Hyper Text Markup Language. É uma linguagem de marcação usada para estruturar páginas na internet.Ele não é uma linguagem de programação, mas sim a forma de organizar textos, imagens, links e outros elementos para que os navegadores possam exibi-los corretamente.É a base de qualquer site.

Estrutura básica de uma página

Todo o documento HTML segue um padrão.É como um esqueleto que indica onde ficam as informações invisíveis ao usuário (< head >) e onde aparece o conteúdo visível, como texto e imagens (< body >).

Tags e Elementos

As tags são os blocos de construção do HTML. Elas funcionam como instruções que dizem ao navegador o que cada parte do conteúdo representa — se é um parágrafo, um título, uma lista ou uma imagem. Elas geralmente têm uma abertura (< p >) e um fechamento (< /p >), e o conteúdo fica entre elas.

Voltar

Títulos e Parágrafos

O HTML organiza a hierarquia de textos com títulos e parágrafos. Essa divisão facilita a leitura, ajuda a dar destaque ao que é mais importante e cria uma experiência de navegação mais clara. Os níveis dos títulos vão do < h1 > ao < h6 >, sendo o < h1 > o título principal, e os parágrafos são representados com a tag < p >.

Links e Imagens

Com HTML, é possível conectar páginas entre si através de links (usando a tag < a >) e tornar o conteúdo mais visual com imagens (usando a tag < img >). Esses recursos são fundamentais para a web ser interativa e atrativa.

Voltar

Listas

As listas servem para organizar informações de forma prática e direta. Elas podem ser ordenadas, quando há uma ordem a seguir, ou não ordenadas, quando apenas se deseja listar itens.
